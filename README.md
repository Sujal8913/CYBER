# CYBER

#🤖 CYBER – AI Chat Assistant

CYBER is an intelligent AI chat assistant similar to ChatGPT. It supports real-time conversations,anytype of problem solving
with simple interface and a basic chat interface. CYBER is designed to help users solve real-life problems, answer questions,
and assist with productivity.

🚀 Features

💬 Real-time AI chat interface

🧠 AI-powered responses using LLM APIs ( Gemini )

🎨 Basic UI 

🔐 Secure backend API integration

🌐 Deployable on web servers
